# 📸 Gallery App

A responsive and interactive Gallery App built using **HTML, CSS, and JavaScript** in a single file. The application showcases images in an attractive gallery layout with smooth animations and a user-friendly design.

## 🚀 Features

* Responsive gallery layout
* Interactive hover effects
* Smooth animations and transitions
* Clean and modern UI
* Mobile-friendly design
* Built entirely in a single HTML file

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```text
Gallery-App/
│
└── index.html
```

## 🎯 Project Purpose

This project was created to practice and improve front-end development skills, including:

* HTML structure and semantic elements
* CSS styling and responsive design
* JavaScript interactivity
* User interface design

## 💡 How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Explore the gallery and its interactive features.
